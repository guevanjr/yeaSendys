API Integration Guide

Your webapp receives API requests from external applications and responds with a pre-filled form if the user exists, or an empty form if not.

API Endpoint:

POST /api/user-form

Expected Request Format:

{
  "userId": "12345",
  "sourceApp": "external-app"
}

Response Format (User Exists):

{
  "status": "success",
  "userExists": true,
  "userData": {
    "name": "John Doe",
    "email": "john.doe@example.com",
    "phone": "+1 (555) 123-4567",
    // ... other fields
  },
  "formUrl": "https://your-webapp.com/form?prefill=true&userId=12345"
}

Response Format (New User):

{
  "status": "success",
  "userExists": false,
  "formUrl": "https://your-webapp.com/form?prefill=false"
}

Node.js/Express Backend Example:

app.post('/api/user-form', async (req, res) => {
  const { userId, sourceApp } = req.body;
  try {
    // Check if user exists in database
    const user = await User.findById(userId);
    if (user) {
      // User exists - return pre-filled form URL
      res.json({
        status: 'success',
        userExists: true,
        userData: user,
        formUrl: `https://your-webapp.com/form?prefill=true&userId=${userId}`
      });
    } else {
      // New user - return empty form URL
      res.json({
        status: 'success',
        userExists: false,
        formUrl: 'https://your-webapp.com/form?prefill=false'
      });
    }
  } catch (error) {
    res.status(500).json({
      status: 'error',
      message: 'Internal server error'
    });
  }
});
